# Project Summary
AMC Alarmas y Monitoreo del Caribe, SRL is enhancing its professional website to provide comprehensive security and monitoring services. The platform showcases alarm systems, video surveillance, and various monitoring options. It features a user-friendly interface with contact forms and social media connections. Recent updates have focused on improving visual appeal, functionality, and user engagement, including a permanent dark mode and streamlined service information.

# Project Module Description
The project consists of several functional modules, including:
- **Header**: Navigation with a company logo and emergency contact information.
- **Footer**: Company details and additional contact methods.
- **ServiceCard**: Display of individual services with WhatsApp contact options, now featuring silhouettes and brief descriptions.
- **ContactForm**: Forms for quote requests and general inquiries, updated to reflect available services.
- **Main Page (Index)**: The main landing page featuring services, company information, and prominent contact options.
- **HexagonBackground**: Animated hexagonal background for visual enhancement.

# File Description Inventory
- **README.md**: Overview of the project and setup instructions.
- **index.html**: Main entry point for the web application.
- **src/App.tsx**: Main React component that initializes the application.
- **src/components/**: Contains all reusable UI components.
- **src/pages/**: Contains page components for routing.
- **src/hooks/**: Custom hooks for managing state and effects.
- **src/lib/**: Utility functions and translation management.
- **src/components/ContactForm.tsx**: Updated component for handling user inquiries and quote requests.
- **src/components/HexagonBackground.tsx**: Component for animated hexagon background.

# Technology Stack
